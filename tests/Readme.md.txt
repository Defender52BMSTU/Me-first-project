Информация о проекте: Проект был создан на основе предыдущей лабораторной работы №5, где мы подбирали коэффициенты для корректирующего устройства. Цель проекта - это получение навыков оформления проектов в Matlab и  правильной структуризации своей работы на примере лабораторной работы №5
Структура проекта:
1. tests - папка в которой находятся тесты модели с разными входными сигналами.
2. scrirts - папка в которой хранятся скрипты для инициализации проекта (запуск констант при открытии) и папка со скриптами для деинициализации проекта (очистка рабочего экрана при закрытии проекта).
3. models - исходная модель люльки с двигателем и редуктором из предыдущей лабораторной.
4. data - константы.
Были созданы 3 Shortcut:
1. Запускает константы.
2. Инициализация проекта.
3. Деинициализация проекта.
Тесты:
Было создано два теста, в ходе которых было поданы на вход сигналы Step и Ramp. Тесты показываю реакции на различные входные воздействия.
